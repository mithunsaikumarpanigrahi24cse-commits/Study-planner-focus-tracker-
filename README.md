# Study Planner & Focus Tracker

A frontend web application that helps students plan daily study tasks and track focused study time using a built-in timer.

## Overview

The Study Planner & Focus Tracker is a simple browser-based tool designed for students to:
- Organize daily study tasks
- Mark tasks as completed
- Measure actual focus time during study sessions

The project is built using only HTML, CSS, and JavaScript and runs entirely on the client side.

## Features

- Add study tasks with:
  - Subject
  - Topic
  - Estimated time (in minutes)
  - Priority (High / Medium / Low)
- View all tasks under “Today’s Tasks”
- Mark tasks as **Completed** (with visual indication)
- Delete tasks when no longer needed
- Focus timer with:
  - Start, Stop, and Reset controls
  - Supports long study durations (HH:MM:SS format)
- Data is stored using  browser localstorage (client-side only)
- Clean and simple user interface

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

## How to Run the Project

1. Clone or download the repository
2. Open the `index.html` file in any modern web browser
3. Add tasks and use the focus timer to track study sessions

_No additional setup or dependencies required._

## Use Case

This project is useful for:
- Students managing daily study schedules
- Tracking completed vs pending tasks
- Monitoring focused study time without distractions
